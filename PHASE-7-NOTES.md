# Phase 7 — Recovery Tank™ Signature Experience

## Included

- Interactive public Recovery Tank™ simulator on the home and pricing pages.
- Business-type selector with adaptive labels, assumptions, and default values.
- Smooth animated transitions between business scenarios and calculation changes.
- Starter Recovery Themes: Vordali, Emerald, Royal, and Rose.
- Animated liquid, waves, particles, bubbles, glow, glass reflections, break-even marker, and counting revenue.
- Annual revenue at risk, protected revenue, estimated net gain, and value-to-cost metrics.
- Recovery Tank™ / Powered by Commit plaque.
- Restored circular Vordali logo in the site header.
- Pro and Enterprise theme definitions are included in the component model for future account-level feature gating. Gold remains Enterprise-only.
- Reduced-motion accessibility support.

## Product-tier intent

- Starter: one tank style, all standard animation features, four Recovery Themes.
- Pro: alternate tank styles and expanded theme library.
- Enterprise: all Pro options plus Gold and custom branding.

## Deployment

1. Replace the existing repository contents with this package.
2. Preserve the production environment variables already configured in Vercel.
3. Deploy from the repository root.
4. No database migration is required for this phase.
